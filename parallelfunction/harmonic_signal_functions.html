Harmonic Wave and Spectrum (f(x), g(x))

Use this tool to explore the harmonic function and its spectrum based on numeric values of f(x) (amplitude) and g(x) (base frequency). You can plug in results from any letter’s system.

Input: f(x) and g(x)

In your notation, f(x) is usually mapped to amplitude A and g(x) to base frequency in Hz. Enter those values here and choose how many harmonics you want to include.

Example: For your S, x = 3 case you might use A ≈ 0.81818 and f ≈ 234.78 Hz with N = 5 harmonics.

Harmonic Wave h(t)

The harmonic function combines N cosine waves at integer multiples of the base frequency: h(t) = Σn=1..N (A / n) · cos(2π n f t)
